— 03 / The character Duck Master ● 03 / 15
— The duck

One duck,
one universe.

Every progression card, ad placement, and reward animation in Duck Master starts from the same 3D mascot — sailor outfit, navy collar, red bowtie, anthropomorphic but unmistakably duck. The brief locked the character; the work was extending it consistently into hundreds of contexts.

01Stay recognisable. Same silhouette, same eye, same beak across every variant.
02Stay playful. Telegram users scroll fast — every variant has to read on a 64-px thumbnail.
03Stay extensible. Each variant is a starting point for a whole job, district, or campaign.
Character — Foundation for every visual
